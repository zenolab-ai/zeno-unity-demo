# Unity Demo for Zeno Platform

Checkout the [introduction](https://github.com/zenolab-ai/zeno-intro) for introduction.
And it should work with latest ARKit supported iOS devices, and ARCore supported Android devices.
We are still polishing the codes, and rapid changing. Please stay stune.


## Demos
### Gravity
Simply drop shapes in an anchored zone. Digital shapes collide on physical space.

![gravity](https://github.com/user-attachments/assets/9d38bcd9-7ec7-45d7-a3af-0893599d53b1)


### LittleSoldier
Spawn little soldiers and obstacles in an anchored zone. The soldiers automatically find path and navigate through in physical space following a ray cursor where the phone is pointing to.

![littlesoldier](https://github.com/user-attachments/assets/f1e54d7b-7468-4e8c-b726-64dd37ddb762)

 
## Setup
Use Unity 6000.2.7f2 to open the project. You can either build with individual demo scene, or main scene to have all demos in one app.

Please report issues if you cannot build or run this project.


## Get Access
Zeno SDK requires API key and secret to access Zeno's cloud service for phenomenal zone preparing.

For try out and demo purpose, please contact developer@zenolab.ai for the API key and secret, and fill out the AppAuthOptions asset in the Unity project.
